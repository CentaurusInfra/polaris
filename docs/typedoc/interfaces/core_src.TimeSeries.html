Interface TimeSeries<T>

Represents a sequence of measurements, i.e., samples, of a single metric.

The samples begin at the time denoted by start and are ordered chronologically until the time denoted by end.

Type parameters

  • T

    The TypeScript type used to represent the data in the samples.
